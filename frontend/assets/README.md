# Assets Directory

This directory contains assets for the theGarage React Native app:

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon foreground
- `favicon.png` - Web favicon
- `splash.png` - Splash screen image
- `fonts/` - Font files (Inter)

## theGarage Branding
- Primary color: #ff6b35 (warm orange)
- Secondary color: #0f172a (ocean blue)
- Font: Inter

These assets maintain theGarage's dual-perspective branding across all platforms.
